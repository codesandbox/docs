# Pricing

CodeSandbox SDK is based on two main components:

- VM credits (used to measure minutes of runtime of VMs created with the SDK)
- Rate limit tier (in case you require rate limits beyond our default)

## VM credits pricing

We price VM credits depending on the size of the VM being run.
